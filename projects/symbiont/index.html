Project Name: Symbiont
Current Openings for new Contributors: 2 more Software Developers required as at 15th January 2023
Description:

Symbiont is a social media platform that has a chrome extension that allows anyone to comment in a message thread leading off any word on any website in the world.

It will also have crypto functionality to link to the UTU trust network which allows for endorsements of information using UTU coin.

The platform uses self censorship. People subscribe to not see comments which have been tagged by a certain number of people. For example if someone is vegan they could subscribe to NOT see any comment which 1000 people have tagged with the tag "meat".
